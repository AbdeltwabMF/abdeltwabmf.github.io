Programs and Equipment I Use

I'm only putting here programs I consider tried and have used for a while.

Software

Basic

Operating system

Artix Linux which is a fork of Arch Linux except for without systemd.

Terminal

st (simple terminal) by suckless.org, which is one of the most minimal, yet easily customizable terminal emulators out there.

Shell

zsh as a shell. It has pretty much all of the features of bash with additional plugins for more advanced auto-completion and syntax coloring.

Window manager

dwm (dynamic window manager). The status bar I use for dwm is dwmblocks.

Text editor

Vim.

IDE

Emacs.

Web browser

Brave, aka Le Shill Lion. Probably the only browser that doesn't require the significant addition of plugins to make it functional. Blocks ads and trackers by default and comes with anti-fingerprinting abilities and Tor windows.

Utilities

File manager

nnn. It's a full-featured terminal file manager. written in C Language.

Mail client

mu4e [1] [2] [3] [4] [5] (mu for emacs). I keep all my mail offline with isync.

Audio player

mpd + ncmpcpp + mpc combination for a library, mpv for playing audio manually.

Video player

mpv. Don't bother with any other video player.

RSS reader

newsboat. You can simply give accounts' or sites' feeds to newsboat and watch videos remotely via mpv without having to open a browser.

Torrent client

Transmission. with the transmission-remote-cli as an interface.
